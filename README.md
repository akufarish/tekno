## How to use

> first Clone this project

```
git clone https://github.com/Agita-Tech/akukreatif-web.git
```

> then run composer install

```
composer install
```

> run npm install `require node js installed`

```
npm install
```

> configure .env file

```
cp .env.example .env
change database setting
```

> run this command

```
php artisan key generate
php artisan migrate
php artisan db:seed
```
